# Netpulse Packet Capture

## Description
Netpulse is a cutting-edge packet capture tool designed for analyzing network traffic and ensuring secure data transmission. Ideal for developers and network administrators, it provides deep insights into packet-level data with an intuitive interface.

## Features
- Real-time packet capturing
- Filter options for specific protocols and ports
- User-friendly graphical interface
- Supports IPv4 and IPv6 protocols
- Ability to export captured data in various formats

## Installation
To install Netpulse, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/asadullahjvd/Netpulse_packetcapture.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Netpulse_packetcapture
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To run the application, use the following command:
```bash
npm start
```  
Visit `http://localhost:3000` to access the interface.
